Steps to deploy code on Github Pages

You can also deploy your github Repositry on the Github pages .

1. Make a Repositry on github.
2. Upload your code in the Repositry
3. Click on the Setting.
4. Click on pages option on left side Navigation.
5. Select your branch from which you want to deploy your code .
6. Select Root Folder and click on save button.
7. Now click on auto generated Url.
8. Your project is running on the url.
